Виртуальное окружение
В папке проекта:
python -m venv venv
venv\Scripts\activate


Установка пакетов
pip install django selenium webdriver-manager


Создание проекта
django-admin startproject CatalogProject
cd CatalogProject
python manage.py startapp catalog


Миграции
python manage.py makemigrations
python manage.py migrate
rm db.sqlite3 (удалить БД)


Суперпользователь
python manage.py createsuperuser


Запуск проекта
python manage.py runserver


Запуск тестов (старая версия)
python manage.py test catalog

Запуск тестов (новая версия)
python manage.py test catalog.tests.test_routes
python manage.py test catalog.tests.test_content
python manage.py test catalog.tests --verbosity=2
